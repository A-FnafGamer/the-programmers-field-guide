## Activity 1 (Performing Tasks)

Discover  the key to getting started with programming. Computers are unintelligent; software empowers these machines to perform great tasks.  

## Activity 2 (Build Your First program)

The first task in this unit is to build a program. You will learn how to write, compile, and run programs using .NET Core and the C# programming language. 

## Activity 3 (Creating and using objects)

Discover how to create and use objects in your C# programs. Discover how to create some windows, draw images, and play sounds using SplashKit. 

## Activity 4 (Wrap Up)

Let’s recap what you have covered this week, and look at the tasks you should aim to have finished before next week.
