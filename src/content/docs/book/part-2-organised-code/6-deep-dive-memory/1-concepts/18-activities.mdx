## Activity 1 (Introduction)

Start to explore the C# library and think about how objects work in other programming languages.

## Activity 2 (Making the most of C#)

Examine the structure of the C# class library, and start to think about how you can make use of this to build larger programs.

## Activity 3 (Other Languages)

See how the concepts we have learnt in C# apply in other programming languages such as python.

## Activity 4 (Objects and Memory)

Examine the lower level details of how objects work so that you can understand how different strategies other languages use.

## Activity 5 (Wrap Up)

Reflect on what you have learnt this week, and examine the tasks to complete in OnTrack.
