Object Oriented development concepts are often transferable between programming languages. See how we can write part of the Light Simulator program in Python 3.

Let's use the concepts we have learnt to take a look at the `LightBulb` class from the Light Simulator program in Python. After watching the video, consider that while syntax changes across languages, the concepts often do not.
