---
title: Time for a boost!
---

See how to make use of logical operators to allow the user to use different keys to move the ship, and to add a booster with a secret key code.

Logical operators allow you to join conditions using **or** and **and**. In this video we will show you how to use both of these operators to allow the user to use the arrow keys or WASD keys to control the ship, and add a secret key code to fire the boosters and engage *ludicrous mode*.
